# ПЕРЕЗАПУСК СЕРВЕРА

## Сервер был остановлен

Теперь запустите сервер заново:

```bash
dotnet run
```

Или если используете другую команду для запуска.

## После запуска сервера:

1. Откройте браузер в **режиме инкогнито**: `Ctrl + Shift + N`
2. Перейдите на `http://localhost:8888`
3. Откройте DevTools: `F12`
4. Вкладка **Network** → поставьте галочку **Disable cache**
5. Перезагрузите страницу
6. Проверьте, что `orders.css?v=20241202001` загружается со статусом 200

## Если после перезапуска все равно не работает:

Сообщите мне:
1. Загружается ли файл `orders.css?v=20241202001` в Network?
2. Какой статус у файла?
3. Есть ли ошибки в Console?

# Добавленные функции

## ✅ Что добавлено:

### 1. Комментарии к заказу
Пользователи могут оставлять комментарии при оформлении заказа.

**Как работает:**
1. При оформлении заказа появляется prompt с полем для комментария
2. Комментарий сохраняется в базе данных
3. Комментарий отображается в деталях заказа (для пользователя и админа)

**Примеры использования:**
- "Доставка после 18:00"
- "Позвоните за час до доставки"
- "Нужна накладная на юр. лицо"

### 2. Просмотр деталей заказа
Добавлена кнопка "Детали" для каждого заказа.

**Что показывается:**
- Номер заказа
- Дата и время
- Статус
- Сумма
- Комментарий (если есть)
- Список товаров с ценами
- Кнопка "Повторить заказ" (для завершённых)

### 3. Остатки на складе (уже было)
Система проверяет наличие товара при оформлении заказа.

**Как работает:**
- При добавлении в корзину проверяется Stock
- При оформлении заказа Stock уменьшается
- Если товара нет - показывается ошибка

---

## 🔧 Что нужно сделать:

### 1. Выполните SQL скрипт

```sql
-- Добавляет поле Comment к Orders
ALTER TABLE "Orders" ADD COLUMN IF NOT EXISTS "Comment" TEXT;
```

Или выполните файл:
```bash
psql -U postgres -d toolshopdb -f add_order_comment.sql
```

### 2. Перезапустите приложение

```bash
# Ctrl+C
dotnet run
```

### 3. Проверьте работу

1. Добавьте товары в корзину
2. Нажмите "Оформить заказ"
3. Введите комментарий (или оставьте пустым)
4. Нажмите OK
5. Перейдите в "Мои заказы"
6. Нажмите "Детали" на заказе
7. Проверьте что комментарий отображается

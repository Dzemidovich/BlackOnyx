# Чеклист безопасности

## ✅ Сделано автоматически:

- [x] Rate Limiting (защита от брутфорса)
  - Глобально: 100 req/min
  - Вход: 5 попыток/15 мин
  - Регистрация: 3 попытки/час

- [x] Валидация входных данных
  - Email формат
  - Пароль минимум 8 символов
  - Пароль содержит буквы и цифры

- [x] JWT токены
  - Срок жизни: 2 часа (было 24)
  - Уникальный ID токена (JTI)
  - Поддержка переменных окружения

- [x] CORS
  - Dev: только localhost
  - Prod: только указанные домены
  - Строгие методы и заголовки

- [x] Security Headers
  - X-Content-Type-Options: nosniff
  - X-Frame-Options: DENY
  - X-XSS-Protection: 1; mode=block
  - Content-Security-Policy
  - Referrer-Policy

- [x] XSS защита
  - Функция escapeHtml() в JavaScript
  - Экранирование пользовательского ввода

- [x] Password Hashing
  - BCrypt с автоматической солью

## 🔧 Нужно сделать СЕЙЧАС:

- [ ] Сгенерировать JWT Secret Key
  ```powershell
  .\generate_jwt_key.ps1
  ```

- [ ] Обновить appsettings.json
  ```json
  "SecretKey": "ваш_новый_ключ"
  ```

- [ ] Перезапустить приложение
  ```bash
  dotnet run
  ```

- [ ] Протестировать
  - [ ] Регистрация работает
  - [ ] Вход работает
  - [ ] Слабый пароль отклоняется
  - [ ] После 5 неудачных попыток входа - блокировка

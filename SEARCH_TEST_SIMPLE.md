# Тест Поиска По Всем Страницам

## Ваша Ситуация
- **На странице**: ~50 товаров
- **Всего страниц**: 200+
- **Всего товаров**: ~10,000+

## Что Исправлено ✅

### БЫЛО (неправильно):
- Поиск работал только по текущей странице (50 товаров)
- Если товар на странице 150, его не находило

### СТАЛО (правильно):
- Поиск работает по **ВСЕМ 10,000+ товарам**
- Находит товары с **ЛЮБОЙ страницы** (1, 50, 150, 200+)
- При поиске показывает **ВСЕ найденные результаты** на одной странице

## Технические Детали

### Запрос БЕЗ поиска (пагинация):
```
GET /api/products?page=1&pageSize=40
→ Возвращает 40 товаров со страницы 1
```

### Запрос С поиском (все результаты):
```
GET /api/products?search=дрель&page=1&pageSize=999999
→ Возвращает ВСЕ товары с "дрель" из всех 10,000+
```

## Что Изменилось в Коде

### Файл: `wwwroot/js/modules/products.js`

**БЫЛО:**
```javascript
const pageSize = search ? 1000 : this.productsPerPage;
params.append('page', search ? 1 : this.currentProductPage);
params.append('pageSize', pageSize);
```
❌ Ограничение 1000 товаров при поиске

**СТАЛО:**
```javascript
if (search) {
    params.append('page', 1);
    params.append('pageSize', 999999); // Все результаты
} else {
    params.append('page', this.currentProductPage);
    params.append('pageSize', this.productsPerPage);
}
```
✅ Без ограничений при поиске

### Файл: `Controllers/ProductsController.cs`

**БЫЛО:**
```csharp
query = query.Where(p =>
    p.Name.Contains(searchTerm) ||
    p.Article.Contains(searchTerm));
```
❌ Чувствительно к регистру

**СТАЛО:**
```csharp
var lowerSearchTerm = searchTerm.ToLower();
query = query.Where(p =>
    p.Name.ToLower().Contains(lowerSearchTerm) ||
    p.Article.ToLower().Contains(lowerSearchTerm));
```
✅ Регистронезависимый поиск

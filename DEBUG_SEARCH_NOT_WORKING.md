# Диагностика: Поиск не работает

## Что нужно проверить

### 1. Откройте DevTools (F12)

1. Нажмите **F12** в браузере
2. Перейдите на вкладку **Network** (Сеть)
3. Введите что-то в поиск (например, "дрель")
4. Найдите запрос к `/api/products?search=...`

### 2. Проверьте URL запроса

**Должно быть:**
```
/api/products?search=дрель&page=1&pageSize=40
```

**Если видите:**
- `pageSize=999999` - кэш не очищен, нажмите Ctrl+Shift+R
- Запроса вообще нет - JavaScript не работает

### 3. Проверьте Response (ответ сервера)

Кликните на запрос → вкладка **Response**

**Что должно быть:**
```json
[
  {
    "id": 1,
    "name": "Дрель электрическая",
    "article": "DR-001",
    ...
  },
  ...
]
```

**Если видите:**
- `[]` (пустой массив) - сервер не находит товары
- `500 Internal Server Error` - ошибка на сервере
- `401 Unauthorized` - проблема с авторизацией

### 4. Проверьте Response Headers

**Должны быть заголовки:**
```
X-Total-Count: 150
X-Page-Size: 40
X-Current-Page: 1
X-Total-Pages: 4
```

**Если заголовков нет:**
- Проблема в контроллере
- Сервер не перезапущен

### 5. Проверьте Console (консоль)

Перейдите на вкладку **Console**

**Ищите ошибки:**
- Красные сообщения об ошибках
- `Uncaught TypeError` - ошибка в JavaScript
- `CORS error` - проблема с CORS

---

## Возможные причины

### Причина 1: Сервер не перезапущен
**Решение:**
```bash
# Остановите (Ctrl+C)
dotnet run
```

### Причина 2: Кэш не очищен
**Решение:**
Нажмите **Ctrl + Shift + R** на странице приложения

### Причина 3: EF.Functions.ILike не работает
**Решение:**
Я уже откатил изменения - теперь используется `ToLower().Contains()`

### Причина 4: База данных пустая
**Решение:**
Проверьте что в базе есть товары:
```sql
SELECT COUNT(*) FROM "Products" WHERE "IsActive" = true;
```

### Причина 5: Проблема с авторизацией
**Решение:**
Проверьте что вы авторизованы (есть токен в localStorage)

## Быстрая проверка

### Тест 1: Проверка API напрямую

Откройте в браузере:
```
http://localhost:ВАШІ_ПОРТ/test_search_api.html
```

Введите поисковый запрос и нажмите "Search".

**Если работает:**
- API работает правильно
- Проблема в основном приложении (JavaScript)

**Если не работает:**
- Проблема в API (контроллер)
- Проблема в базе данных

### Тест 2: Проверка без поиска

1. Откройте страницу товаров
2. НЕ вводите ничего в поиск
3. Товары должны отображаться

**Если товары отображаются:**
- Проблема только с поиском

**Если товары НЕ отображаются:**
- Проблема с загрузкой товаров вообще

## Что делать дальше?

### Если API работает (test_search_api.html находит товары):

1. Проблема в JavaScript основного приложения
2. Проверьте консоль браузера на ошибки
3. Убедитесь что кэш очищен (Ctrl+Shift+R)

### Если API не работает (test_search_api.html не находит товары):

1. Проблема в контроллере или базе данных
2. Проверьте логи сервера
3. Проверьте что в базе есть товары

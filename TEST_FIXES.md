# Тестирование исправлений багов

## Перед тестированием

1. **Перезапустите сервер** (если он запущен)
2. **Очистите кеш браузера**: `Ctrl + Shift + R` (Windows) или `Cmd + Shift + R` (Mac)
3. Или откройте сайт в режиме инкогнито: `Ctrl + Shift + N`

## Тест 1: Кнопка "Оформить заказ" в корзине

### Шаги:
1. Войдите как пользователь (не администратор)
2. Перейдите в каталог товаров
3. Добавьте несколько товаров в корзину
4. Откройте корзину
5. Нажмите кнопку "Оформить заказ"

### Ожидаемый результат:
✅ Открывается модальное окно с:
- Заголовком "Оформление заказа"
- Информацией о количестве товаров и сумме
- Полем для ввода комментария (необязательно)
- Кнопками "Отмена" и "Оформить заказ"

### Если не работает:
- Откройте консоль браузера (F12 → Console)
- Проверьте наличие ошибок JavaScript
- Убедитесь, что файл `wwwroot/js/modules/auth.js` загружается

## Тест 2: Кнопка "Детали заказа" для администратора

### Шаги:
1. Войдите как администратор (Admin или Manager)
2. Перейдите на вкладку "Заказы"
3. Найдите любой заказ в списке
4. Посмотрите на кнопку в столбце "Действия"

### Ожидаемый результат:
✅ Кнопка называется "Детали заказа" (не просто "Детали")
✅ При нажатии открывается модальное окно с:
- Заголовком "Заказ #[номер]"
- Основной информацией (дата, статус, сумма)
- **Информацией о клиенте** (это новое!):
  - Для физ. лиц: ФИО и Email
  - Для юр. лиц: название компании, УНП, контактное лицо, телефон, адреса, банковские реквизиты
- Списком товаров в заказе

### Если не работает:
- Откройте консоль браузера (F12 → Console)
- Проверьте наличие ошибок JavaScript
- Убедитесь, что вы вошли как администратор (role === 'Admin' или 'Manager')

## Тест 3: Обычный пользователь НЕ видит регистрационные данные

### Шаги:
1. Войдите как обычный пользователь (не администратор)
2. Перейдите на вкладку "Заказы"
3. Нажмите кнопку "Детали" около любого заказа

### Ожидаемый результат:
✅ Кнопка называется просто "Детали" (не "Детали заказа")
✅ В модальном окне НЕТ раздела "Информация о клиенте"
✅ Показывается только:
- Дата, статус, сумма
- Email пользователя
- Список товаров

## Проверка изменений в файлах

Если ничего не работает, проверьте, что изменения применились:

### Файл: wwwroot/js/modules/auth.js
Строка 7 должна быть:
```javascript
showModal(modalId, content = null) {
```

### Файл: wwwroot/js/modules/orders.js
Строка ~97 должна содержать:
```javascript
const isAdmin = this.currentUser && (this.currentUser.role === 'Admin' || this.currentUser.role === 'Manager');
```

Строка ~113 должна содержать:
```javascript
<i class="fas fa-eye"></i> ${isAdmin ? 'Детали заказа' : 'Детали'}
```

## Отладка через консоль браузера

Откройте консоль (F12) и выполните:

```javascript
// Проверка текущего пользователя
console.log('Current user:', app.currentUser);

// Проверка функции showModal
console.log('showModal function:', app.showModal.toString());

// Проверка, что функция принимает 2 параметра
console.log('showModal parameters count:', app.showModal.length); // Должно быть 1 (второй параметр необязательный)
```

## Если проблема сохраняется

1. Полностью закройте браузер и откройте снова
2. Попробуйте другой браузер
3. Проверьте, что сервер перезапущен
4. Проверьте консоль на наличие ошибок 404 (файлы не найдены)

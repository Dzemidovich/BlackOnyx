# Bugfix Requirements Document

## Introduction

Кнопки управления скидками в админ-панели программы лояльности не реагируют на нажатия. HTML-код использует inline обработчики событий вида `onclick="app.openDiscountModal()"`, но объект `app` не доступен в глобальной области видимости `window`, что приводит к ошибкам JavaScript при попытке вызова методов.

Проблема затрагивает следующие кнопки в разделе управления пользователями:
- Кнопка "Скидка" (вызывает `app.openDiscountModal()`)
- Кнопка "История скидок" (вызывает `app.openDiscountHistoryModal()`)
- Другие кнопки, использующие аналогичный паттерн вызова методов через `app.*`

Файл `wwwroot/js/app.js` определяет объект `app` как локальную переменную (`var app = {...}`), а файл `wwwroot/js/pages/admin.js` расширяет его через `Object.assign(app, {...})`. Однако нигде в коде не выполняется присваивание `window.app = app`, что делает объект недоступным для inline обработчиков событий в HTML.

## Bug Analysis

### Current Behavior (Defect)

1.1 WHEN администратор нажимает кнопку "Скидка" в списке пользователей THEN браузер выбрасывает ошибку "ReferenceError: app is not defined" и модальное окно не открывается

1.2 WHEN администратор нажимает кнопку "История скидок" в списке пользователей THEN браузер выбрасывает ошибку "ReferenceError: app is not defined" и модальное окно не открывается

1.3 WHEN HTML-код пытается вызвать любой метод через `onclick="app.method()"` THEN JavaScript не может найти объект `app` в глобальной области видимости

1.4 WHEN файл `app.js` загружается THEN объект `app` создается как локальная переменная и не привязывается к `window`

### Expected Behavior (Correct)

2.1 WHEN администратор нажимает кнопку "Скидка" в списке пользователей THEN система SHALL открыть модальное окно для изменения скидки пользователя без ошибок JavaScript

2.2 WHEN администратор нажимает кнопку "История скидок" в списке пользователей THEN система SHALL открыть модальное окно с историей изменений скидок без ошибок JavaScript

2.3 WHEN HTML-код вызывает метод через `onclick="app.method()"` THEN JavaScript SHALL успешно найти объект `app` в глобальной области видимости и выполнить метод

2.4 WHEN файл `app.js` загружается THEN объект `app` SHALL быть привязан к `window.app` для доступа из inline обработчиков событий

### Unchanged Behavior (Regression Prevention)

3.1 WHEN существующий код обращается к `app` напрямую (без `window.`) внутри JavaScript модулей THEN система SHALL CONTINUE TO работать корректно

3.2 WHEN другие страницы и компоненты используют методы объекта `app` THEN они SHALL CONTINUE TO функционировать без изменений

3.3 WHEN выполняется навигация между страницами приложения THEN система SHALL CONTINUE TO корректно загружать и отображать контент

3.4 WHEN пользователь взаимодействует с другими кнопками и элементами интерфейса THEN они SHALL CONTINUE TO работать как ожидается

3.5 WHEN загружаются модули через `Object.assign(app, {...})` THEN они SHALL CONTINUE TO корректно расширять объект `app`
